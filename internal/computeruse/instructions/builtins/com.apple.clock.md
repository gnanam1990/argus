# Clock

- The window has four tabs in the sidebar: World Clock, Alarm, Stopwatch, and Timer. Click a tab before acting; actions on one tab (e.g. "add alarm") do nothing if Timer or World Clock is active.
- To add an alarm: select the Alarm tab, click the "+" button in the top-right, set the time in the picker, then click Save.
- To start a timer: select the Timer tab, set duration with the scroll wheels, then click Start. A running timer shows a Pause/Cancel pair in place of Start.
- The Stopwatch tab has Start, then Stop/Lap once running; Reset only appears after Stop.
- World Clock cities are added via the "+" button and searched by city name; there is no edit, only add/delete.
- Prefer clicking the specific tab first, then locating controls, rather than searching the whole window for a button label that may exist under a different tab.
