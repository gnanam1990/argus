# Calculator

- Basic mode is the default small window. Switch to Scientific (Cmd+2) for trig/log/exponent keys, or Programmer (Cmd+3) for bitwise and base conversion; the button grid changes shape with each mode, so re-check button positions after switching.
- Keyboard input works directly: digits, +, -, *, /, and Return for "=" all map to calculator keys, which is usually more reliable than clicking small buttons.
- "C" clears the current entry; "AC"/holding clear resets the whole calculation. There is no undo.
- The result display at the top is read-only text, not an editable field; do not attempt to click into it to fix a typo, clear and retype instead.
- Scientific mode's "2nd"/Inverse toggle changes several button labels (e.g. sin to sin⁻¹); confirm the toggle state before clicking a trig key.
