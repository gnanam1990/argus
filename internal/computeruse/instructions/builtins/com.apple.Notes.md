# Notes

- Cmd+N creates a new note in the currently selected folder. Only press it once; a double-send creates two empty notes.
- The left sidebar lists folders (e.g. iCloud, Notes, Recently Deleted). Click a folder first to scope where a new note lands, then create it.
- The note list (middle column) can be hidden on narrow windows; if search or selection by list item fails, widen the window or use the folder view toggle in the toolbar.
- The first line typed in a note becomes its title automatically; there is no separate title field.
- Checklists, tables, and formatting live under the toolbar's format ("Aa") menu, not in a menu bar item.
- Search (top of sidebar) filters across all folders by default; scope to a folder by clicking it before searching if results are noisy.
